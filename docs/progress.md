# OneResume 进展文档

## 2026-03-28

### 阶段 0：项目骨架
- 已初始化 `Vue 3 + TypeScript + Vite` 项目。
- 已接入 `vue-router`、`pinia`、`tailwindcss`、`markdown-it`、`unified`、`remark-parse`、`idb`、`vuedraggable@next`、`lucide-vue-next` 等核心依赖。
- 已补齐基础测试、类型检查和构建命令，保证后续迭代可验证。

### 阶段 1：本地数据层
- 已完成简历领域模型、示例数据和 schema 迁移入口。
- 已完成 `localStorage` 的工作区索引与简历文档持久化。
- 已完成 `IndexedDB` 的头像资源存储与引用读取。
- 已完成 `JSON / Markdown` 的基础导入导出能力。

### 阶段 2：MVP 工作台
- 已完成顶部工具栏、多简历切换、新建、重命名、删除、重置与清空本地数据。
- 已完成左侧结构化编辑器，支持基础信息、栏目显隐、重命名、条目增删改和拖拽排序。
- 已完成右侧 A4 实时预览，支持多模板、样式调整和缩放。
- 已完成打印路由与浏览器打印导出 PDF 闭环。
- 已完成头像上传、裁剪、本地持久化与移除。

### 阶段 3：验证与构建
- 已通过 `npm run test`。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 4：Markdown / HTML 导入兼容
- 已识别并兼容“Markdown 后缀但实际包裹 HTML 简历”的导入情况。
- 已扩展导入器，支持在 Markdown 中自动识别 HTML 内容并分流解析。
- 已补充中文栏目标题映射，兼容“教育背景 / 实习经历 / 项目经历 / 专业技能”等常见命名。
- 已补充对应测试，覆盖基础信息、教育、经历、项目与技能解析。

### 阶段 5：导入导出与界面收敛
- 已将工具栏整理为统一的“导入 / 导出”入口。
- 已支持按文件类型自动导入 `HTML / HTM / Markdown / TXT / JSON`。
- 已支持导出 `PDF / Markdown / 工程文件`，当来源是 HTML 时额外支持导出原始 HTML。
- 已新增 HTML 简历的“原模板预览 / 结构化预览”切换模式。
- 已将“专业技能”模块改为轻量文本输入，不再显示时间、地点、标题等不合适字段。

### 阶段 6：HTML 原模板预览修复
- 已将 HTML 原模板预览从 `v-html` 改为 `iframe srcdoc` 隔离渲染，避免站内样式污染导入模板。
- 已同步调整打印链路，HTML 原模板模式下按 iframe 内文档进行打印。
- 已增加本地图片路径提示，明确浏览器对磁盘图片访问的限制。

### 阶段 7：HTML 原模板少显示修复
- 已让 iframe 根据模板真实内容尺寸自适应，不再固定死 A4 容器。
- 已加入 `ResizeObserver + 图片 load/error 监听 + requestAnimationFrame` 的尺寸同步机制。
- 已修复缩放后底部裁切、占位高度不足和少显示问题。
- 已完成本轮 `test / typecheck / build` 验证。

### 阶段 8：工作台高度对齐与面板化
- 已将工作台布局改为 `flex column + main flex-1`，避免依赖脆弱的 `calc(100vh-xxx)`。
- 已为左侧编辑区和右侧预览区引入统一的 `workbench-panel` 面板外壳，统一圆角、边框、阴影和内部滚动区域。
- 已将 HTML 本地图片提示收回到右侧预览面板内部，避免再次破坏左右高度对齐。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 9：头像配置与自定义模块闭环
- 已新增 `photoPlacement`，支持头像在简历中的左右位置切换，并兼容旧数据迁移。
- 已增强头像上传链路，仅接受图片文件，支持更换与移除。
- 已调整基础信息区的头像编辑卡片布局，让头像操作与个人信息表单分区更清晰。
- 已补齐自定义模块闭环：新增时自动生成首个内容块，支持编辑标题、说明、Markdown 内容，并支持删除。
- 已修复自定义模块在预览中重复显示副标题元信息的问题。
- 已通过 `npm run test`。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 10：同步预览入口
- 已为 HTML 原模板模式补充“同步预览”入口，顶部工具栏和左侧提示卡都可一键切换。
- 已补充明确提示文案，说明右侧会跟随左侧修改实时更新。
- 已通过 `npm run test`。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 11：上传 HTML 模板同步渲染
- 已新增 HTML 模板同步器，在保留上传模板原始样式骨架的前提下，把结构化编辑后的姓名、联系方式、头像和模块内容回填到模板中。
- 已新增 `source-html-sync` 预览模式，“同步预览”不再切回站内默认模板，而是继续使用上传的 HTML 模板实时更新。
- 已打通打印链路，上传模板同步预览状态下导出 PDF 也沿用同步后的 HTML 模板内容。
- 已新增对应测试，覆盖上传 HTML 模板的姓名、联系方式、项目、技能和头像同步。
- 已通过 `npm run test`。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 12：模块设置表单化
- 已为每个模块补充独立设置表单，支持直接修改模块名称。
- 已新增模块类型下拉，可在教育背景、工作经历、项目经历、专业技能和自定义模块之间切换，切换后条目表单随模块类型变化。
- 已新增模块顺序下拉，可直接移动到指定位置，同时保留拖拽排序能力。
- 已将“新增自定义模块”入口收敛成更通用的“新增模块”。
- 已通过 `npm run test`。
- 已通过 `npm run typecheck`。
- 已通过 `npm run build`。

### 阶段 13：头像比例调整
- 已将头像上传裁剪从默认 `1:1` 改为常见二寸证件照比例 `35:49`。
- 已统一左侧编辑卡片与右侧结构化简历模板中的头像展示比例，避免上传后和预览时比例不一致。
- 已补充头像裁剪比例单元测试，覆盖宽图和长图的居中裁剪场景。

### 阶段 14：顶部工具栏收敛与标签页图标
- 已移除顶部品牌宣传文案、缩放按钮组和工具栏里的头像快捷入口，减少视觉噪音。
- 已整理顶部工具栏与工作台页面中的中文文案，清理这一轮涉及区域的乱码文本。
- 已新增极简 SVG favicon，并接入页面标题与站点描述。

### 阶段 15：品牌区优化与文本加粗
- 已优化左上角品牌区，升级为图标加字标的一体化展示，整体更像产品工作台而不是占位标签。
- 已为个人简介、项目描述、工作经历描述、专业技能、自定义模块内容补充“选中文字一键加粗”能力，底层采用 Markdown `**加粗**` 包裹。
- 已补充文本包裹工具测试，并让预览区支持渲染个人简介中的 Markdown 加粗效果。

### 阶段 16：导出链路修正
- 已重写 Markdown 导出逻辑，结构化简历按当前模块顺序、显示状态和编辑内容导出。
- 已让 HTML 简历在同步预览模式下先按当前模板同步结果渲染，再转换为 Markdown，避免导出内容落回旧格式。
- 已让 Markdown 导出携带头像图片数据，导出后的文件不再丢头像。
- 已补强打印视图，导出 PDF 时会把页面标题设置为当前简历名，并在打印前等待图片加载，减少头像丢失。

### 阶段 17：Markdown 头像兼容修正
- 已将 Markdown 导出中的头像从超长 `data:` 图片链接语法改为内联 HTML `<img>` 标签，提升 Typora 等编辑器的兼容性。
- 已同步覆盖结构化导出和 HTML 同步模板导出两条链路，避免头像再次被拆成普通文本。

### 阶段 18：导出入口收口
- 已按当前上线需求移除产品里的 Markdown 导出入口，避免用户继续走到未最终定稿的导出格式。
- 已同步收敛工具栏导出事件分支，当前保留 `PDF / 工程文件 / 原始 HTML` 三类导出能力。

### 阶段 19：Git 上传准备
- 已补充适合 `Vue 3 + Vite + TypeScript` 项目的 `.gitignore`，新增环境变量、构建缓存、覆盖率、测试产物和常见系统文件忽略规则。
- 已确认当前目录尚未初始化 Git 仓库，后续上传步骤需从 `git init` 开始。
